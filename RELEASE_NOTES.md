# 一键三线表插件 1.1.0

发布时间：2026-06-08

## 发布类型

GitHub Pages + DMG 分发版。适合 Mac 用户从 GitHub Releases 下载 `.dmg` 后直接安装 Word 加载项。

## 核心功能

- 将选中的 Word 表格一键转换为科研论文常用三线表。
- 自动按窗口宽度调整表格。
- 更彻底地清除旧表格框线，包括单元格直接框线和段落边框。
- 更彻底地去除底纹，包括表格、单元格、段落和文字层底纹。
- 默认整体上/下框线为 `0.75 pt` 黑色单线。
- 默认首行下框线为 `0.5 pt` 黑色单线。
- 支持分别设置中文字体和西文字体。
- 在 Word for Mac/Windows 中自动读取 Word 当前可用字体列表。
- 支持设置字号、字体颜色和首行加粗。
- 支持保存多个模板，点击模板即可套用。

## Mac 使用方式

1. 下载 `three-line-table-word-1.1.0-github.dmg`。
2. 打开 DMG。
3. 双击 `一键三线表插件安装器.app`。
4. 完全退出并重新打开 Microsoft Word。
5. 在 Word 的“开始 > 加载项”中打开“一键三线表插件”。

## 说明

插件页面由 GitHub Pages 托管，用户电脑不需要安装 Node，也不需要保持终端服务运行。

当前 DMG 未使用 Apple Developer ID 签名和 notarization。若 macOS 提示来自未认证开发者，请右键点击安装器并选择“打开”。
